# Actuator and OpenAPI API

These endpoints are generated by Spring Boot Actuator and springdoc-openapi. Security configuration permits health, info, Prometheus, Swagger UI, and OpenAPI docs without JWT. Other exposed actuator endpoints require JWT unless security is changed.

## Public Endpoints

### GET `/actuator/health`

Application health.

**Auth:** Public

**Request body:** none

**cURL**

```bash
curl -X GET 'http://localhost:9500/actuator/health'
```

**Response**

```json
{
  "status": "UP",
  "groups": [
    "liveness",
    "readiness"
  ]
}
```

### GET `/actuator/info`

Application build/git/environment info.

**Auth:** Public

**Request body:** none

**cURL**

```bash
curl -X GET 'http://localhost:9500/actuator/info'
```

**Response**

```json
{
  "app": {
    "name": "lambrk-backend"
  }
}
```

### GET `/actuator/prometheus`

Prometheus metrics scrape endpoint.

**Auth:** Public

**Request body:** none

**cURL**

```bash
curl -X GET 'http://localhost:9500/actuator/prometheus'
```

**Response**

```text
# HELP http_server_requests_seconds HTTP server request duration
# TYPE http_server_requests_seconds summary
...
```

### GET `/swagger-ui/index.html`

Interactive Swagger UI.

**Auth:** Public

**Request body:** none

**cURL**

```bash
curl -X GET 'http://localhost:9500/swagger-ui/index.html'
```

**Response**

HTML page for Swagger UI.

### GET `/v3/api-docs`

Full OpenAPI document.

**Auth:** Public

**Request body:** none

**cURL**

```bash
curl -X GET 'http://localhost:9500/v3/api-docs'
```

**Response**

```json
{
  "openapi": "3.1.0",
  "info": {
    "title": "Lambrk Backend API",
    "version": "1.0.0"
  },
  "paths": {}
}
```

### GET `/v3/api-docs/{group}`

Group-specific OpenAPI document. Groups include `public`, `auth`, `posts`, `comments`, `users`, `communities`, `votes`, `search`, `notifications`, `admin`, `recommendations`, `files`, and `actuator`.

**Auth:** Public

**Path parameters**

| Name | Type | Required | Description |
| --- | --- | --- | --- |
| `group` | string | yes | OpenAPI group name. |

**cURL**

```bash
curl -X GET 'http://localhost:9500/v3/api-docs/posts'
```

**Response**

```json
{
  "openapi": "3.1.0",
  "info": {
    "title": "Lambrk Backend API",
    "version": "1.0.0"
  },
  "paths": {}
}
```

## Protected Actuator Endpoints

`application.yml` exposes `metrics`, `env`, `configprops`, `loggers`, `threaddump`, and `heapdump`. These are matched by the default authenticated rule.

### GET `/actuator/metrics`

List metric names.

**Auth:** User/Admin JWT

**cURL**

```bash
curl -X GET 'http://localhost:9500/actuator/metrics' \
  -H 'Authorization: Bearer <token>'
```

**Response**

```json
{
  "names": [
    "http.server.requests",
    "jvm.memory.used"
  ]
}
```
